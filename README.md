# 🛡️ TheSJArchive Filter List

**TheSJArchive Filter List** is a custom, curated adblock filter list designed for [Brave](https://brave.com/), [uBlock Origin](https://ublockorigin.com/), and [AdGuard](https://adguard.com/).

It combines blocking rules, safe unblocking of trusted services, and cosmetic filters to create a smoother, privacy-friendly browsing experience without breaking important site features.

---

## ✅ Features

- 🚫 Blocks ads, trackers, popups, malicious or suspicious domains
- ✅ Unblocks essential services (Google Fonts, Adobe, Cloudflare, etc.) so sites work correctly
- 🎨 Includes site-specific cosmetic rules to hide unwanted elements
- 🛠️ Easy to read, update, and customize for personal use

---

## 💻 How to Use in Brave

1. Click on **Raw** in GitHub to get the direct link to this filter list file.
2. Open **Brave** and go to:

Settings → Shields → Custom Filters → Custom Filter Lists

3. Click **Add a New Filter List** and paste the RAW URL.
4. Save and **Update** the list.

Brave will automatically apply these filters to your browsing. You can also manually **Update** the list in settings at any time.

---

## 🌐 Works With

- Brave Shields
- uBlock Origin
- AdGuard (desktop and browser extensions)

## 📜 Philosophy
